# PDF 版权页扫描与清理工具 v2.0

## 功能概述

PDF 版权页扫描工具是一个智能 PDF 批量处理桌面软件，可以：

1. **OCR 扫描版权页**：通过 OCR 识别指定页面范围的版权关键词
2. **自动删除版权页**：检测到版权页后自动删除
3. **自动删除空白页**：基于文本内容智能检测并删除空白页
4. **文件备份**：修改前自动备份原文件
5. **断点续扫**：中断后可继续，自动跳过已处理文件
6. **并发处理**：支持多文件并行扫描和页面级 OCR 并发

## 界面说明

启动软件后，主窗口包含以下区域：

### 1. 参数设置面板

- **源目录**：待处理的 PDF 文件夹路径
- **备份目录**：原文件备份存放路径（可选）
- **关键词**：每行一个关键词，支持 AND/OR 搜索逻辑
- **检查页面**：要检查的页面范围字符串
  - `"2"` - 只检查第2页
  - `"2:5"` - 检查第2到5页
  - `"2,-1"` - 检查第2页和最后一页
  - `"2:5,7:10,-3:-1"` - 组合多个范围
- **DPI**：PDF 渲染分辨率，建议 300
- **文件并发**：同时处理的文件数量（默认 4）
- **OCR 并发**：同一文件内多页面并行 OCR 数量（默认 2）

### 2. OCR 设置

- **OCR 模式**：百度 / 火山 / 讯飞 / 本地 Tesseract
- **识别精度**：general_basic（通用）/ accurate_basic（高精度）
- **语言**：chi_sim（中文）/ eng（英文）等

### 3. 选项

- **删除版权页**：发现版权页后自动删除
- **删除空白页**：自动检测并删除空白页
- **区分大小写**：关键词大小写敏感
- **过滤空格**：匹配时忽略空格
- **模糊匹配**：容许少量干扰字符
- **调试模式**：输出 OCR 识别详情

### 4. 控制按钮

- **开始扫描**：启动扫描流程
- **暂停**：暂停当前扫描（可在文件间隙暂停）
- **继续**：恢复暂停的扫描
- **停止**：终止扫描并保存当前进度
- **保存配置**：将当前设置保存到 settings.json

### 5. 实时日志区

显示带颜色区分的运行日志：
- 白色：一般信息
- 黄色：警告
- 红色：错误
- 绿色：成功

### 6. 扫描结果列表

表格展示每个文件的处理结果：
- 文件名
- 状态（已修改 / 未修改 / 失败 / 已跳过）
- 版权页列表
- 空白页删除数量
- 总页数
- 处理耗时
- 详情信息

## 配置方式

### 方式一：GUI 设置（推荐）

直接在软件界面中设置参数，点击「保存配置」即可持久化。

### 方式二：settings.json

非敏感配置保存在 `settings.json`（开发模式）或 `%APPDATA%/PDFScanner/settings.json`（打包后）。

### 方式三：.env 环境变量（OCR 密钥）

在项目目录创建 `.env` 文件，填入 OCR 服务密钥：

```env
# 百度 OCR
BAIDU_APP_ID=your_app_id
BAIDU_API_KEY=your_api_key
BAIDU_SECRET_KEY=your_secret_key

# 火山引擎 OCR
VOLC_ACCESS_KEY=your_access_key
VOLC_SECRET_KEY=your_secret_key

# 科大讯飞 OCR
IFLYTEK_APP_ID=your_app_id
IFLYTEK_API_KEY=your_api_key
IFLYTEK_SECRET_KEY=your_secret_key
```

## 安装与运行

### 开发环境运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置 OCR 密钥（复制模板并编辑）
copy .env.example .env
# 用文本编辑器编辑 .env 填入密钥

# 3. 启动 GUI
python main.py
```

### 打包为 exe

```bash
# 1. 安装打包依赖
pip install pyinstaller

# 2. 执行打包脚本
python build.py

# 3. 输出文件
# dist/PDFScanner.exe
```

打包后，将 `.env` 文件放在 `PDFScanner.exe` 同级目录，或设置系统环境变量。

## 依赖说明

| 依赖 | 用途 |
|------|------|
| PyMuPDF | PDF 渲染、文本提取、页面删除（替代 PyPDF2 + pdf2image + poppler）|
| Pillow | 图像处理 |
| PySide6 | GUI 界面框架 |
| pydantic-settings | 配置管理 |
| pytesseract | 本地 OCR（可选）|
| baidu-aip | 百度 OCR（可选）|
| volcengine | 火山 OCR（可选）|
| requests | 讯飞 OCR 网络请求（可选）|

## 技术升级说明（v1 -> v2）

v2.0 相比原命令行版本的主要改进：

1. **统一 PDF 引擎**：使用 PyMuPDF 替代 PyPDF2 + pdf2image + poppler，无需额外安装 poppler，渲染速度提升 3-5 倍
2. **图形界面**：PySide6 现代化界面，操作直观
3. **并发处理**：多文件并行 + 页面级 OCR 并发，大幅缩短批量处理时间
4. **暂停/继续**：扫描过程中可随时暂停和恢复
5. **配置管理**：支持 GUI 设置、JSON 配置文件、环境变量三种方式
6. **安全性**：移除代码中的硬编码 API 密钥，改用环境变量
7. **单文件打包**：支持打包为独立 exe，无外部依赖

## 断点续扫

扫描进度自动保存在 `pdf_scan_progress.json`（开发模式）或 `%APPDATA%/PDFScanner/pdf_scan_progress.json`（打包后）。

进度文件结构：

```json
{
  "scanned_files": ["文件1.pdf", "文件2.pdf"],
  "modified_files": ["文件1.pdf"],
  "unmodified_files": ["文件2.pdf"]
}
```

重新启动扫描时，会自动跳过已处理的文件。

## 注意事项

1. **首次运行前**：
   - 确保已安装依赖：`pip install -r requirements.txt`
   - 配置好 OCR 服务密钥（`.env` 文件）
   - 建议先在小批量文件上测试

2. **备份安全**：
   - 修改前自动备份原文件到备份目录
   - 每个文件只备份一次（原始完整文件）
   - 不修改的文件不会生成备份

3. **页码说明**：
   - 正数表示从前往后（1 表示第 1 页）
   - 负数表示从后往前（-1 表示最后一页）
   - 支持连续范围：`"2:5"` 表示第 2 到 5 页
   - 支持多个范围组合：`"2:5,7:10,-3:-1"`

4. **关键词匹配**：
   - `AND`：所有关键词都要匹配
   - `OR`：匹配任意一个关键词即可
   - 支持模糊匹配（容许少量干扰字符）

5. **调试排查**：
   - 勾选「调试模式」查看 OCR 识别详情
   - 根据识别结果调整关键词或 DPI
   - OCR 识别不到文字时，尝试提高 DPI 或切换高精度模式
